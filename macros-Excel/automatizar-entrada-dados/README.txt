Cria formulário para acrescentar, editar e apagar linhas. Este Macro é dividido em 2 módulos e 3 formulários.

    Crie formulário principal: Salve arquivo como .xlsm/No Visual Basic Editor: Insert UserForm/Crie botões (novo, editar, deletar, fechar) e listbox para mostrar dados. Para editar: Selecione elemento/View.
    Crie planilha auxiliar (Planilha_Aux) com colunas contendo os valores (sem repetição) das colunas que terão lista suspensa.
    Crie dois módulos: modRange (funções auxiliares) e modMain.
    Crie formulário para inserir nova linha: crie botões (salvar, fechar), OptionButtons (para selecionar entre opções; ex. M ou F) e caixas de texto para preenchimento dos campos e seleção de uma lista suspensa.
    Crie formulário para editar uma linha: crie botões (atualizar, fechar), OptionButtons (para selecionar entre opções; ex. M ou F) e caixas de texto para preenchimento dos campos e seleção de uma lista suspensa.
